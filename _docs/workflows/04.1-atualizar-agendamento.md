# 04.1 [EXTRA] Atualizar Agendamento

> **Type:** n8n Flow
> **Trigger:** webhook (called when client or nutritionist reschedules)
> **Status:** active

## Short Description

Updates or reschedules an existing appointment in Google Calendar and the NutrIA-Pro database, handling cancellation of the old slot and creation of the new one with client notification.

## Responsibilities

- Receive rescheduling request with old and new appointment details
- Authenticate with Google Calendar using tenant's OAuth credentials
- Delete or update the existing Google Calendar event
- Create new event at the rescheduled time
- Update appointment record in NutrIA-Pro database
- Re-schedule appointment reminders for the new time
- Notify client of the rescheduling via Chatwoot/WhatsApp

## Integration Points

- **Google Calendar API** — event update/delete and re-create
- **Chatwoot API** — rescheduling notification to client
- **NutrIA-Pro Backend API** — update appointment record, reschedule reminders
- **Secretária v3 (01)** — may be called as sub-workflow

## Implementation Decision

**Stay in n8n.** Low volume, event-driven, external API calls. Rescheduling logic has multiple conditional branches well-suited to visual flow.
